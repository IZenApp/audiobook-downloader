## 🚀 Установка и запуск

### Быстрый старт
```bash
git clone https://github.com/YOUR_USERNAME/audiobook-downloader.git
cd audiobook-downloader
./scripts/install.sh
./scripts/launcher.sh
```

### Настройка
```bash
# Создайте свой список книг
cp data/books_example.txt data/books.txt
# Отредактируйте data/books.txt под свои нужды
```
